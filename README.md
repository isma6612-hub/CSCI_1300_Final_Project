# CSCI 1300 Final Project

## Theme

The theme of this game is the Legend of Zelda. The player can travel to regions from the franchise, fight monsters, and enter dungeons.

## Goal

The player will have to find the pieces of the triforce and fight Ganon before three days end.

## How to compile and run



## How to play

Go to shops to buy materials, go to dungeons to get the required pieces of the Triforce, and go to the castle to fight Ganon

## Classes

Fighter:
Has a name, health, and power that is used to calculate if it dies and how much damage it does to others

Player:
Inherits the characteristics of the fighter, but also has money and an inventory that can contain object of the item class
